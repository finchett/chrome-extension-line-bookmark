# Simple pins
https://github.com/user-attachments/assets/394a9ccc-74ad-4d4c-9e05-337265c95b1b

**Quickly mark and navigate long documents.**

* `Alt + left click`: Add a pin.
* `Ctrl + X`: Cycle through pins.
* Click a pin: Remove it.
